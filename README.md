# Email Reader Website

A modern, responsive email client web application that allows users to read, organize, and manage their emails through an intuitive interface.

## Features

### 📧 Email Management
- **Inbox View**: Read and organize incoming emails
- **Folder Organization**: Separate folders for Inbox, Sent, Drafts, Trash, and Spam
- **Email Composition**: Write and send new emails
- **Reply & Forward**: Respond to emails or forward them to others
- **Email Actions**: Delete, mark as read, select multiple emails

### 🎨 User Interface
- **Clean Design**: Modern, professional interface
- **Responsive Layout**: Works on desktop, tablet, and mobile devices
- **Unread Indicators**: Visual cues for unread emails
- **Email Preview**: See sender, subject, and message preview
- **Search Functionality**: Easy navigation through email folders

### ⚡ Interactive Features
- **Real-time Updates**: Instant email status changes
- **Modal Compose Window**: Clean email composition interface
- **Email Selection**: Multi-select emails for batch operations
- **Keyboard Navigation**: Intuitive navigation controls

## Getting Started

1. **Open the Website**: Simply open `index.html` in any modern web browser
2. **Browse Emails**: Click on folder names in the sidebar to navigate between email folders
3. **Read Emails**: Click on any email in the list to read its full content
4. **Compose Email**: Click the "Compose" button to write a new email
5. **Email Actions**: Use the action buttons to reply, forward, or delete emails
